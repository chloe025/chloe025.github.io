Bienvenue sur mon portefolio !

Je suis Chloé ZANNIER, j'ai 19 ans et je suis actuellement étudiante en 2ème année de DUT Informatique à l'IUT de Metz, en Lorraine.
Ce site a été réalisé dans le cadre d'un devoir noté mais j'en avais déjà imaginé et créé une grande partie en vue d'appliquer mes différents apprentissages en informatique.
Le but de ce portefolio est de vous présenter et mes projets réalisés durant mon cursus scolaire (pré et post baccalauréat) et de vous proposer mon Curriculum vitæ. Vous trouverez donc toutes les informations sur mes réalisations ou moi-même dans les différents menus.


Je l'ai codé en HTML5, CSS 3 et j'ai utilisé pour la première fois Bootstrap.
Pourquoi Bootstrap ?
J'ai décidé d'utiliser cet outil dans le but de découvrir de nouvelles fonctionnalités et d'avoir un site responsive qui peut correspondre à tous les types d'appareils. De plus, il permet de grandement faciliter mon travail, tout en restant pratique et assez facile d'utilisation.

Cependant, il reste encore beaucoup de choses à faire et améliorer, donc il risque de souvent changer.
Choses à terminer et/ou améliorer :

  • Rechercher le code redondant et améliorer le CSS
  • Améliorer le responsive
  • Améliorer le code pour les lecteurs d'écrans
  • Inclure du JavaScript et/ou Ajax
