Projet Java : Boutique de Noël

Le premier projet de 2ème année fut celui de CPOA. Nous avons appronfondi les notions de Java fut au deuxième semestre et avons utilisé des modèles comme celui du MVC avec les DAO.

Le projet :
Le but de ce projet était de développer une application de gestion de boutique simple. Grâce aux DAO, nous devions pouvoir facilement basculer du mode MySql à une liste mémoire pour travailler en local. Nous devions pouvoir ajouter/modifier/supprimer des catégories, produits, clients et commandes sans erreurs. Le tri par différents choix était aussi demandé.
Cependant, nous avons rencontré beaucoup de difficultés durant ce projet (notamment avec JavaFx où l'interface aurait pu être améliorée).

Ce que cela m'a apporté :
La boutique de Noël fut assez compliquée pour moi -ayant peu d'affinités avec le langage Java- mais j'ai pu découvrir et mettre en place le modèle MVC (que j'ai essayé de réutiliser en place dans le projet php à la page suivante). Le modèle des DAO et les méthodes agiles ont aussi été utilisées afin d'avoir un code beaucoup plus lisible et optimisé.
Néanmoins, à l'exception de 2, toutes les fonctionnalités demandées étaient opérantes.
Ce projet m'a demandé de nouvelles méthodes de travail, plus efficaces que je m'efforce de mettre en place dans les projets actuels.

Répartition du travail :
Mon binôme a effectué une plus grande part de travail que moi. J'ai résolu une partie des bugs et j'ai écrit les fonctions de la liste mémoire. J'ai créé la page d'accueil et mis en place le début des Controllers.
